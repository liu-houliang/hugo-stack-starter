## Welcome

This is an open-source beautified version based on **Hugo Theme Stack v4**. This template aims to provide an out-of-the-box blog solution.

### Why choose this template?

- **Partially Beautification**: Based on the official theme for partial modification.
- **Modern**: Developed based on the latest v4 version.
- **Easy to Customize**: All configurations are modular.

### Start Writing

You can directly modify the Markdown files under the `content/post` directory to start your creation.

```markdown
---
title: "My New Post"
date: 2026-04-09
---
This is content.
```
